# 2 Mins - Product Requirements Document

Version: 1.0

---

# Goal

Help users immediately start a tiny action that takes less than two minutes.

The app should remove the friction of starting.

---

# MVP Features

## 1. Daily Suggestions

Users receive a curated list of tiny actions.

Examples:

- Drink water
- Stretch
- Delete 10 photos
- Organize desktop
- Reply to one message
- Take deep breaths

The app should never overwhelm the user.

Only a small number of suggestions should be visible.

---

## 2. Categories

Each task belongs to one category.

Examples:

- Mind
- Body
- Home
- Digital
- Social
- Productivity
- Self Care

---

## 3. Random Task

Users can request another suggestion.

The next suggestion should not immediately repeat previous ones.

---

## 4. Complete Task

A task can be completed with a single tap.

Completion should feel rewarding.

No long celebration.

A subtle animation is enough.

---

## 5. History

Users can view previously completed tasks.

History should remain simple.

No statistics overload.

---

## 6. Favorites

Users can save favorite tasks.

Favorites should appear in a dedicated section.

---

## 7. Search

Users can search tasks.

Search should be instant.

---

## 8. Notifications

Daily reminders.

Fully customizable.

Examples:

08:30

13:00

20:00

Notifications should be friendly.

Never guilt users.

---

## 9. Offline Support

Everything works offline.

No account required.

---

## 10. Dark Mode

Support:

Light

Dark

System

---

# Future Features

Not included in MVP.

- AI generated tasks
- Widgets
- Apple Watch
- WearOS
- Siri Shortcuts
- Cloud Sync
- Shared tasks
- Premium Packs

---

# Non Functional Requirements

App launch under 2 seconds.

Smooth 60 FPS animations.

Responsive UI.

Accessible.

Battery friendly.

Offline first.

---

# Success Criteria

The user should be able to:

Open app.

Find a task.

Complete it.

Close app.

All within two minutes.

---

# Out of Scope

No Pomodoro timer.

No habit tracking.

No social feed.

No chat.

No leaderboard.

No ads.
